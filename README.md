# Fundamentos de Desarrollo Web

## Resumen del curso

Este curso introduce los fundamentos del desarrollo web para estudiantes de secundaria de 12 a 17 anos sin experiencia previa. La propuesta esta organizada en seis lecciones en video, con una duracion total aproximada de 60 minutos, y desarrolla los conceptos de forma progresiva hasta crear una pagina web personal usando HTML y CSS.

El curso no requiere instalacion de herramientas. Las practicas se realizan en editores gratuitos en linea, accesibles desde un navegador web como Chrome, Firefox, Safari o Edge.

**Duracion total:** 60 minutos aproximadamente  
**Numero de lecciones:** 6 videos  
**Publico objetivo:** estudiantes de secundaria sin experiencia previa en desarrollo web  
**Prerrequisitos:** ninguno  
**Herramientas:** CodePen, JSFiddle, Replit y GitHub  
**Repositorio de codigo fuente:** [webdev-course-HackRats](https://github.com/Eslander-Celis/webdev-course-HackRats)

## Objetivos de aprendizaje

Al finalizar el curso, los estudiantes seran capaces de:

- Comprender que es un sitio web y como funciona en el navegador.
- Identificar la relacion entre HTML como contenido y CSS como estilo.
- Crear una estructura basica de pagina web con HTML.
- Usar etiquetas comunes como encabezados, parrafos, listas, imagenes y enlaces.
- Aplicar estilos basicos con selectores y propiedades CSS.
- Personalizar colores, fuentes, bordes, espacios y alineacion.
- Combinar HTML y CSS para crear una pagina web sencilla.
- Reconocer buenas practicas y errores comunes para principiantes.

## Leccion 1: Que es el desarrollo web

La primera leccion presenta el desarrollo web como el proceso de crear sitios y paginas que se visualizan desde un navegador. Se explica que una pagina web combina contenido, estructura y estilo.

El estudiante identifica el rol de HTML para organizar el contenido y el rol de CSS para definir la apariencia visual.

**Conceptos principales:**

- Sitio web.
- Pagina web.
- Navegador.
- HTML.
- CSS.
- Editor en linea.

**Actividad:** abrir un editor web en linea y observar la relacion entre contenido HTML y estilo CSS.

## Leccion 2: Introduccion a HTML

La segunda leccion explica la estructura basica de un documento HTML. Se presentan etiquetas esenciales y se crea una primera pagina con titulo y parrafos.

**Conceptos principales:**

- Documento HTML.
- Etiquetas.
- Elementos.
- Atributos.
- Estructura `html`, `head` y `body`.
- Encabezados y parrafos.

**Actividad:** crear una primera pagina HTML con un titulo principal y una breve presentacion personal.

## Leccion 3: Elementos HTML comunes

La tercera leccion amplia el uso de HTML mediante elementos frecuentes en paginas web. Se agregan listas, imagenes y enlaces para enriquecer el contenido.

**Conceptos principales:**

- Listas ordenadas y no ordenadas.
- Imagenes.
- Enlaces.
- Atributos `src`, `alt` y `href`.
- Organizacion semantica basica.

**Actividad:** completar una pagina personal con una lista de intereses, una imagen representativa y un enlace externo.

## Leccion 4: Introduccion a CSS

La cuarta leccion introduce CSS como lenguaje de estilos. Se explica como seleccionar elementos HTML y aplicar propiedades visuales.

**Conceptos principales:**

- Selectores CSS.
- Propiedades y valores.
- Colores.
- Fuentes.
- Bordes.
- Espaciado.
- Alineacion.

**Actividad:** aplicar estilos a una pagina HTML para cambiar colores, fuente, bordes y distribucion visual.

## Leccion 5: Pagina web personal con HTML y CSS

La quinta leccion integra HTML y CSS para crear una pagina web personal sencilla. El estudiante combina estructura, contenido y estilos para producir una pagina visualmente organizada.

**Conceptos principales:**

- Estructura de pagina.
- Secciones de contenido.
- Tarjetas simples.
- Estilos reutilizables.
- Depuracion de errores comunes.

**Proyecto:** crear una pagina de perfil personal con nombre, descripcion, lista de intereses, imagen, enlaces y estilos personalizados.

## Leccion 6: Buenas practicas y proximos pasos

La sexta leccion presenta recomendaciones para continuar aprendiendo desarrollo web. Se revisan errores comunes, validacion basica del codigo y recursos confiables como MDN Web Docs.

**Conceptos principales:**

- Validacion de HTML.
- Organizacion de archivos.
- Nombres claros de clases.
- Accesibilidad basica.
- Documentacion web.

**Actividad:** revisar la pagina final, corregir errores comunes y preparar una version final para compartir.

### Resumen de elaboracion

El equipo HackRats desarrollo una ruta de aprendizaje introductoria para estudiantes de secundaria, organizada en seis lecciones progresivas. La propuesta inicia con el concepto de sitio web, continua con HTML y CSS, y finaliza con la creacion de una pagina web personal.

Para la elaboracion del curso se definieron los temas de cada leccion, se prepararon ejemplos HTML/CSS, se seleccionaron herramientas gratuitas en linea, se organizaron los guiones y se estructuraron los materiales en un repositorio publico de GitHub.
