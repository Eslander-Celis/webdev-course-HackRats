¿Qué es el desarrollo web?

El desarrollo web consiste en crear páginas y sitios que se pueden ver desde un navegador.

HTML

HTML organiza el contenido de una página web, como títulos, párrafos, listas, imágenes y enlaces.

CSS

CSS cambia la apariencia visual de la página, como colores, fuentes, tamaños y espacios.

En resumen, una página web combina contenido y estilo.
