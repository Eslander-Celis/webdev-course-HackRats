Listas en HTML: <ul>, <ol> y <li>

En HTML usamos listas para agrupar elementos relacionados. Existen principalmente dos tipos: listas no ordenadas (<ul>) y listas ordenadas (<ol>). Cada elemento de la lista se escribe con <li>.

Ejemplo: lista no ordenada (<ul>)

Útil cuando el orden no importa (por ejemplo, una lista de frutas):

  • Manzana
  • Pera
  • Uva

Ejemplo: lista ordenada (<ol>)

Usa <ol> cuando el orden sí importa (pasos, instrucciones, ranking):

  1. Primero
  2. Segundo
  3. Tercero

Ejemplo: lista de tecnologías

Podemos añadir clases para aplicar estilos con CSS:

  • HTML
  • CSS
  • JavaScript

Ejemplo: pasos (ordenados)

Las listas ordenadas pueden usar atributos como start o reversed:

  1. Abrir editor
  2. Escribir código
  3. Guardar archivo

Ejemplo: lista anidada

Las listas pueden contener otras listas dentro de un <li> para mostrar subelementos:

  • Frutas
    • Manzana
    • Pera
  • Verduras
    • Zanahoria
    • Lechuga

Buenas prácticas

  • Usa <ul> para listas sin orden y <ol> para listas ordenadas.
  • Siempre coloca cada elemento dentro de <li>.
  • Para accesibilidad, añade aria-label o un encabezado que describa la lista.
