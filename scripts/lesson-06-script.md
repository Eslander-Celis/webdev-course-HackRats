## Desarrollo

### Introducción

En esta lección revisaremos errores comunes en HTML y CSS, aprenderemos cómo encontrarlos y aplicaremos buenas prácticas para organizar mejor el código.

### Errores comunes en HTML

Incorrecto:

```html
<h1>Mi página</h2>
```

Correcto:

```html
<h1>Mi página</h1>
```

También debemos cerrar los párrafos y escribir los atributos entre comillas.

### Errores comunes en CSS

Incorrecto:

```css
h1 {
  color blue
}
```

Correcto:

```css
h1 {
  color: blue;
}
```

### Proceso para encontrar errores

1. Revisar etiquetas de apertura y cierre.
2. Verificar atributos y comillas.
3. Revisar dos puntos y puntos y coma.
4. Comprobar las llaves.
5. Probar cambios pequeños.
6. Comparar con un ejemplo correcto.

### Buenas prácticas

- Usar sangría.
- Escribir nombres claros.
- Agregar `alt` a las imágenes.
- Separar HTML y CSS.
- Revisar el resultado después de cada cambio.
- No copiar código sin comprenderlo.
