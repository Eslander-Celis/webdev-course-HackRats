## 2. ¿Qué es el desarrollo web?

### Narración

El desarrollo web es el proceso de crear páginas y sitios que podemos ver desde un navegador.

Por ejemplo, cuando entramos a una página de noticias, a una tienda en línea, a una red social o a una página de una institución educativa, estamos usando un sitio web.

Un sitio web puede tener texto, imágenes, botones, enlaces, colores y diferentes secciones.

Para crear ese contenido usamos tecnologías web. En este curso nos enfocaremos en dos de las más importantes para empezar: HTML y CSS.

### Texto en pantalla

```text
Desarrollo web = crear páginas y sitios para el navegador
```

---

## 3. HTML y CSS

### Narración

HTML sirve para organizar el contenido de una página web.

Con HTML podemos indicar qué parte será un título, qué parte será un párrafo, dónde irá una imagen o dónde colocaremos un enlace.

CSS sirve para cambiar el estilo visual.

Con CSS podemos modificar colores, fuentes, tamaños, bordes y espacios.

Una forma sencilla de recordarlo es esta:

HTML es el contenido.

CSS es el estilo.

### Texto en pantalla

```text
HTML = contenido
CSS = estilo
```

---

## 4. Ejemplo sencillo

### Narración

Observemos un ejemplo muy simple de HTML.

### Código mostrado

```html
<h1>Mi primera página web</h1>

<p>Estoy aprendiendo desarrollo web.</p>
```

### Narración

La etiqueta `<h1>` crea un título principal.

La etiqueta `<p>` crea un párrafo.

Cuando escribimos este código en un editor como CodePen, el navegador muestra el título y el texto en la vista previa.

---

## 5. Demostración en CodePen

### Narración

Ahora abriremos CodePen desde el navegador.

CodePen tiene diferentes paneles para escribir HTML, CSS y JavaScript.

En esta primera lección usaremos principalmente el panel HTML.

Primero escribimos un título principal.

### Código mostrado

```html
<h1>¿Qué es el desarrollo web?</h1>
```

### Narración

Luego agregamos un párrafo que explique la idea principal.

### Código mostrado

```html
<p>El desarrollo web consiste en crear páginas y sitios que se pueden ver desde un navegador.</p>
```

### Narración

Después agregamos dos subtítulos: uno para HTML y otro para CSS.

### Código mostrado

```html
<h2>HTML</h2>
<p>HTML organiza el contenido de una página web.</p>

<h2>CSS</h2>
<p>CSS cambia la apariencia visual de la página.</p>
```

### Código completo

```html
<h1>¿Qué es el desarrollo web?</h1>

<p>El desarrollo web consiste en crear páginas y sitios que se pueden ver desde un navegador.</p>

<h2>HTML</h2>

<p>HTML organiza el contenido de una página web, como títulos, párrafos, listas, imágenes y enlaces.</p>

<h2>CSS</h2>

<p>CSS cambia la apariencia visual de la página, como colores, fuentes, tamaños y espacios.</p>

<p>En resumen, una página web combina contenido y estilo.</p>
```

### Narración

Al escribir este código, CodePen muestra el resultado automáticamente.

Esto nos permite ver de inmediato cómo el contenido HTML aparece en el navegador.

## 6. Actividad práctica

### Narración

Ahora es tu turno.

Crea una primera página web usando el siguiente punto de partida.

### Código inicial

```html
<h1>Mi primera página web</h1>

<p>Escribe aquí una frase sobre lo que crees que es un sitio web.</p>

<h2>HTML y CSS</h2>

<p>HTML sirve para organizar el contenido.</p>

<p>CSS sirve para cambiar el estilo visual.</p>
```

### Indicaciones

La página debe incluir:

- Un título principal.
- Un párrafo con una explicación breve sobre los sitios web.
- Un subtítulo sobre HTML y CSS.
- Una frase que explique para qué sirve HTML.
- Una frase que explique para qué sirve CSS.

### Narración

Puedes pausar el video durante tres minutos para completar la actividad.

## 7. Errores comunes

### Error 1: Escribir etiquetas incompletas

#### Incorrecto

```html
<h1>Mi página web
```

#### Correcto

```html
<h1>Mi página web</h1>
```

### Narración

Cuando abrimos una etiqueta, debemos cerrarla correctamente.

---

### Error 2: Confundir HTML con CSS

### Narración

HTML no se usa para decorar visualmente la página. HTML organiza el contenido.

CSS se usa para cambiar la apariencia visual.

Recordemos la idea principal:

HTML es contenido.

CSS es estilo.
